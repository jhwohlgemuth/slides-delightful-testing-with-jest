What is Jest?

A complete and easy to set-up JavaScript testing solution

You can use it to write and run tests and stuff

##Complete solution?

Complete "Batteries Included"

  • "Zero" configuration (most of the time)
  • Built-in code coverage reports (just add --coverage)
  • Robust jasmine-like API
  • Matchers, mocks, spies, stubs, and snapshots
  • Extended with plugins
##Easy set-up?

Easy as "ABC"

A) Install Jest via npm

B) Write tests in __tests__ directory

C) Run tests

Set-up in under a minute
